# PWA-Text-Editor

## Description

This PWA is a single page app text editor that runs in the browser with and without internet connection. This can be used to text notes, edit text, or even make a shopping list.

## Features

You can type in this app to edit text or use text for whatever you need. This app can be downloaded and used offline.
